FAQ

Perguntas Frequentes

Tire suas dúvidas sobre o sistema.

Como funciona o modelo de cobrança?
Trabalhamos com duas opções: Loteamento Digital Estático (pagamento único) e Loteamento com Gestão (setup + mensalidade para acesso ao painel). Entre em contato para conhecer os valores e escolher o melhor plano para você.
O que é o arquivo standalone?
É um arquivo HTML completo e independente com o mapa do seu loteamento. Você pode hospedar no GitHub Pages (gratuito), versel, etc com seu próprio domínio ou utilizar a nossa hospedagem gratuita. Inclui painel administrativo com senha para você gerenciar os status dos lotes.
Preciso de conhecimento técnico?
Não! Você fornece a planta (PDF, DWG ou GeoJSON) e nós criamos tudo. Para atualizar status dos lotes, basta acessar o painel com sua senha — simples como usar WhatsApp.
O mapa funciona offline?
Não. O mapa precisa de internet para carregar as camadas de satélite e sincronizar os dados dos lotes. Mas o consumo de dados é mínimo e otimizado.
Posso personalizar as cores e ícones?
Não. As cores para cada tipo de área (lotes, ruas, áreas verdes, lagos) são padrões, caso tenha uma área diferente das citadas podemos personalizar a cor.
Quantos lotes posso cadastrar?
O valor base cobre até 100 lotes. Para cada 50 lotes adicionais, há um acréscimo no setup. Entre em contato para orçamento personalizado se seu empreendimento for maior.
Os dados ficam seguros?
Sim. Utilizamos o Supabase, uma infraestrutura robusta com criptografia e backups automáticos. Apenas você tem acesso ao painel administrativo através da sua senha exclusiva.
